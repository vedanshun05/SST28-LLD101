# 🚗 Parking Lot Implementation

## 🚀 Implementation
A multilevel system with slot assignment and billing. Below is the file structure:

- `ParkingLot.java`: Central orchestrator for assignment and status.
- `ParkingSlot.java`: Slot entities with gate-based distance maps.
- `ParkingTicket.java`: Entry details tracker.
- `Bill.java`: Exit-based calculation and report.
- `Vehicle.java`, `VehicleType.java`: Vehicle data models.
- `SlotType.java`: Enum containing hourly rates.
- `Main.java`: Simulation with different vehicle types and gates.
